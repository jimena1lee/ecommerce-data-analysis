## 목표

v1 미니멀 페이지를 레퍼런스(topgonyisback.github.io/wedding-card-v2)의 **구동 방식**을
차용해 전면 개편한다. 콘텐츠(10개 섹션, 문구)는 유지하되 디자인·모션을 컬리 브랜드
톤으로 재구축하고, 사용자 제공 증거물 이미지 4장을 추가한다.

## 변경 없는 것

- 파일 위치 `portfolio/index.html` 단일 파일, 주소 동일
- 콘텐츠 내용·순서 (10개 섹션), 전화번호 제외 원칙, iframe 5개 + 브랜드 영상
- 라이브러리 0개 (순수 CSS + 소량 JS)

## 디자인 시스템

- 메인 컬러: 컬리 시그니처 퍼플 `#5f0080`, 틴트 `#f7f0fa`(연보라 배경), 흰 바탕 유지
- 카드: 둥근 모서리(16px) + 부드러운 그림자, 섹션 콘텐츠를 카드 단위로 배치
- 성과 숫자는 퍼플 강조

## 구성 요소 (레퍼런스에서 차용한 구동 방식)

1. **고정 내비게이션** — 상단 고정, 반투명 블러(backdrop-filter). 좌측 "이지원",
   우측 섹션 앵커(적합성·리뷰 분석·경험·프로젝트·AI 역량). 모바일은 가로 스크롤 링크.
2. **풀스크린 히어로** — 100svh, 은은한 퍼플 그라데이션 배경.
   라벨("지원 포지션 · 패션 MD · 컬리") → 헤드라인 "컬리 안에서 패션을 성립시키는 MD"
   (핵심 단어 블러→선명 리베일) → 소개 문장 → 핵심 숫자 칩 3개(경력 8년 · ROAS 37배 ·
   리뷰 1,543건)가 시차 등장 → 하단 스크롤 유도 화살표.
3. **스크롤 리베일** — IntersectionObserver로 `.reveal`에 `.in` 부여.
   opacity 0→1 + translateY(20px)→0, 0.7s, 연속 요소 0.08s 시차.
   이미 뷰포트 안이면 즉시 표시. `prefers-reduced-motion`이면 모션 전부 비활성.
4. **블러 텍스트 리베일** — 히어로 헤드라인 핵심 단어와 마무리 문장
   ("무신사는 고객의 옷장을... 컬리는 냉장고를...")에 blur(10px)→0 전환.
5. **콜아웃 카드** — 프로젝트의 문제=회색 카드, 성과=연보라 카드(퍼플 숫자).
   적합성 3개 강점 = 카드 3장. 경험 표 = 퍼플 틴트 헤더.
6. **지연 로딩 유지** — iframe `loading="lazy"`, 이미지 `loading="lazy"`.
