# 컬리 패션 리뷰 인사이트 대시보드 — 설계

날짜: 2026-07-09
목적: 기존 컬리 리뷰 마이닝 연구(Semantic ID 파이프라인)를 AI 문외한 MD도 쓸 수 있는
인터랙티브 대시보드로 전환. MD 면접 포트폴리오용 — 링크 하나로 공유 가능해야 함.

## 결정 사항 (사용자 합의 완료)

- **형태**: 정적 HTML 파일 1개. 모든 통계·유사도 데이터를 빌드 시점에 미리 계산해
  HTML에 JSON으로 내장. 런타임 AI 호출 없음(비용 0원). GitHub Pages 배포 대상.
- **입력 방식**: 자유 텍스트가 아닌 속성 선택(드롭다운/칩). MD가 이미 아는
  정보(카테고리·계절감·두께 등)를 클릭으로 고르는 방식.
- **빌드**: `kurly/build_dashboard_kurly.py` 스크립트 1개가 `kurly/data/`의 CSV를 읽어
  `kurly/output/dashboard.html`을 생성. 기존 `build_portfolio_kurly_semantic.py`와 같은 패턴.

## 데이터 소스 (모두 기존 파일, 신규 수집 불필요)

| 파일 | 용도 |
| --- | --- |
| `kurly/data/kurly_products_{165,166,169}_20260709.csv` | 상품·브랜드·가격·리뷰수 (288행, 중복 제거 필요) |
| `kurly/data/kurly_attributes_coded.csv` | 리뷰 1,543건 × 8속성 + 감성 + 불만사유 태깅 |
| `kurly/data/kurly_product_semantic_ids.csv` | 상품 199개 Semantic ID·불만비율 |
| `kurly/data/codebook_kurly.json` | 속성 코드북 (핏·사이즈감·기장·비침·두께·신축성·착용상황·계절감·감성) |

## 화면 구성 — 탭 3개

### 탭 1: 신상품 리뷰 미리보기 (메인)

- 입력(좌): 카테고리(필수, 3택1) + 선택 속성: 계절감·두께·핏·신축성·착용상황 + 가격대
- 출력(우, 선택 즉시 갱신):
  1. 유사 상품 카드 — 조건과 속성이 겹치는 기존 상품 목록(상품명·리뷰수·불만율)
  2. 예상 리뷰 프로필 — 유사 상품군의 속성 언급 분포 막대 차트
  3. 리스크 게이지 — 예상 불만율 vs 카테고리 평균, 낮음/보통/높음 3단계
  4. MD 액션 제안 — 규칙 기반 자동 문구(예: 사이즈 작음 리스크 → 상세페이지 안내 권장)
  5. 실제 리뷰 발췌 — 유사 상품에 달린 리뷰 2~3개
- 예측 로직: 실시간 AI 아님. 선택 속성과 상품 Semantic ID 축의 겹침 기반 유사도
  (같은 카테고리 필수, 속성 일치 수로 정렬). 유사 상품 리뷰의 통계를 그대로 보여줌.

### 탭 2: 브랜드 프로필

- 상품 3개 이상 브랜드(약 29개) 드롭다운 선택
- 카드: 상품 수 / 평균 가격·할인율 / 불만율(전체 평균 대비 ▲▼) / 사이즈 경향 /
  주력 착용상황·계절감 / 자주 나오는 불만 사유

### 탭 3: 연구 탐색 보드

- 기존 보고서 4개 차트의 인터랙티브 버전: 속성 언급 빈도(카테고리 필터),
  카테고리별 사이즈감, 불만 유형, 불만율 TOP 상품(클릭 시 실제 불만 리뷰 펼침)
- 하단에 연구 파이프라인 요약(수집→AI 태깅→코드북→Semantic ID) — 면접관용

## 정직성 장치

- 모든 예측 옆에 "유사 상품 N개 · 리뷰 M건 기반" 상시 표기
- 유사 상품 3개 미만이면 예측 대신 "데이터 부족 — 참고용" 안내
- 데이터 규모(상품 199개·리뷰 1,543건)를 대시보드에 명시

## 비기능 요구

- 모바일(폰)에서도 열람 가능 — 면접관이 폰으로 열 수 있음
- 외부 CDN 의존 최소화(오프라인에서도 열리게 차트는 순수 JS/SVG 또는 인라인 라이브러리)
- 한국어 UI. 디자인은 frontend-design·dataviz 스킬 가이드 적용, 컬리 톤(퍼플 계열) 참고

## 검증

- 빌드 스크립트 실행 → dashboard.html 생성 확인
- 브라우저에서 3개 탭 동작·수치 스팟체크(예: 언더웨어 '작게 나옴' 27%,
  불만율 TOP 상품이 기존 리포트와 일치)
- 모바일 뷰포트 확인
